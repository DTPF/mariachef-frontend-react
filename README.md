# Mariachef — Frontend

Recipe sharing platform where users can create, browse and search recipes with image uploads. Built with React and TypeScript.

## Features

- User registration and authentication
- Create and publish recipes with images
- Browse and search recipes
- Category filtering
- User profiles
- Responsive design for mobile and desktop

## Tech Stack

- **Frontend:** React, TypeScript, Ant Design, Sass
- **Auth:** Auth0
- **Routing:** React Router

## Getting Started

1. Clone and set up the backend first — see [mariachef-backend-node](https://github.com/DTPF/mariachef-backend-node)
2. Clone this repo: `git clone https://github.com/DTPF/mariachef-frontend-react.git`
3. Install dependencies: `npm install`
4. Copy `.env.example` and configure your environment variables
5. Start the dev server: `npm start`
